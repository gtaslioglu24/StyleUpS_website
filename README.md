# Outfit AI Web

Animated marketing site + legal pages for App Store and user support.

## Pages

- `/` Landing page
- `/privacy` Privacy Policy
- `/terms` Terms of Use
- `/support` Support
- `/delete-account` Account deletion instructions

## Run Locally

```bash
npm install
npm run dev
```

Then open [http://localhost:3000](http://localhost:3000).

## Suggested Deploy

- Vercel (recommended)
- Netlify
- Firebase Hosting (static export flow)

## App Store Fields

Use these URLs in App Store Connect:

- Privacy Policy URL -> `/privacy`
- Support URL -> `/support`
- Terms of Use URL -> `/terms`
